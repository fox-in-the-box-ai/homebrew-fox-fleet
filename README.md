# homebrew-fox-fleet

Homebrew tap for [Fox Fleet](https://github.com/fox-in-the-box-ai/fox-fleet) — the management plane for Fox AI assistant instances.

## Install

```bash
brew tap fox-in-the-box-ai/fox-fleet
brew install fox-control
```

## Update

```bash
brew upgrade fox-control
```

## Formula

The `fox-control` formula is automatically updated by the fox-fleet release workflow on each stable release.
